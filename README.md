##My Portfolio

### Ruby version
5.0.2

### Set up:

	clone:
	>$ git clone https://github.com/AR8Stefan/my-portfolio.git

	cd intoi the directory:
	>$ cd my-portfolio	

	Bundle the gems:
	>$ bundle

	Start the server:
	>$ rails s
